# SPA-frame
顾名思义，是 SPA（单页应用）的运行框架。

设计理念：
* 通用。框架基于 Web API，可以在任何支持 [Web 规范](https://html.spec.whatwg.org/) 的平台上运行。
* 便携。无论是框架本体还是应用包都只有一个文件，非常便于分发。
* 自包含。应用所需的所有代码、资源均包含在应用包中，应用启动后可以便捷访问任何需要的资源，没有外部依赖，无需进行网络通信。
* 程序驱动范式。不同于一般的网页应用以html文件为入口，本框架下的SPA以主模块为入口。资源、界面构建完全由脚本控制。
